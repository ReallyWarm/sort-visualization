# sort-visualization
Visualize the process of sorting algorithms using Pygame.

## Current features:
- [Button to reset data](https://github.com/ReallyWarm/sort-visualization/blob/master/visualize/Button.py#L19-L40)
- [Button to choose algorithms](https://github.com/ReallyWarm/sort-visualization/blob/master/visualize/Button.py#L43-L126)
- [Button to set data size](https://github.com/ReallyWarm/sort-visualization/blob/master/visualize/Button.py#L131-L204)
